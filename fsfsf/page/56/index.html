3 接口文档

发表于 2021-08-15 分类于 API

实战开发中后端会开发很多接口

每个接口都有大量的参数

思考：前端如果想知道有哪些接口并且每个接口有几个参数？

回答：通过接口文档

思考：接口文档自己写还是怎样？

回答：有很多方法，可通过apiDoc来生成

apiDoc是什么？

就是nodejs中的一个模块

去哪下

https://apidocjs.com/

如何用

步骤1：先下载模块，后期通过命令基于于注释生成文档（下载到全局，一次即可）

步骤2：在项目根目录创建 apidoc.json 文件

步骤3：去写接口注释

步骤4：生成接口文档

apidoc -i 接口注释目录 -o 接口文档存放目录

步骤5：查看doc文件下的index.html
