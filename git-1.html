1 .git文件夹 与 git 对象

发表于 2021-08-15 更新于 2021-08-14 分类于 git

执行git init初始化仓库，进入.git文件夹后

hooks：钩子
info：包含一个全局性排除文件。即哪些文件是Git不用管理的，就放在该文件夹下
objects：目录存储所有数据内容。即数据库
res：目录存储指向数据（分支）的提交对象

config：文件包含项目所持有的配置选项 若在配置用户信息时没有加--global与--system，该文件里会有配置的用户信息
description：用来显示对仓库的描述信息
HEAD：文件指示目前被检出的分支

logs：保存日志
index：文件保存暂存区

git 对象

Git 的核心部分时一个简单的键值对数据库。我们可以向该数据库插入任意类型的内容，它会返回一个键值，通过该键值可以在任意时刻再次检索该内容
