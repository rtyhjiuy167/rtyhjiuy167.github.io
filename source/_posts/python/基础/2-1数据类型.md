<h5>Numbers(数字)</h5>

```python
# Numbers(数字)
#  int(有符整型) long(长整型) float(浮点型) complex(复数)
int1 = 12
print(type(int1))  # <class 'int'>

float1 = 1.1
print(type(float1))  # <class 'float'>

```

<h5>布尔类型</h5>

```python
# 布尔类型 True False
print(type(True))  # <class 'bool'>
```
