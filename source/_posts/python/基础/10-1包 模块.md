<h4>模块</h4>

```python
"""
模块其实是一个.py文件
自定义模块中，通常只是去定义类或函数、变量等等，并不会去调用
如果在自定义模块中，想要写一些测试代码，并以当前模块为主程序来使用执行，
但有不会被导入而执行时，可以把代码写到 if __name__ == '__main__': 下
"""
if __name__ == '__main__':
    print("这个位置写的代码只有当前脚本被直接运行时触发")
"""
导入方法
    1.导入模块的所有内容
        import 模块名 (as 别名)
    2.导入指定模块的指定内容
        from 模块名 import 标识名 (as 别名)
"""
```

<h4>包</h4>

```python
"""
包其实是一个文件夹，里面有一个__init__.py及其它的.py文件
导入方法
    1.导入__init__里配置的模块
        from 包名 import *
        需要配置__init__.py文件
        __init__.py文件里，要有可暴露的模块列表，语法： __all__ = ['模块名']
    2.导入指定包中的指定模块
        from 包名 import 模块名 (as 别名)
        -使用模块名.标识名来使用
    3.导入指定包中的指定模块的指定内容
        from 包名.模块名 import 标识名 (as 别名)
        -可直接使用标识名来使用
"""
```

<h4>导入方式</h4>

```python
"""
1.绝对导入
    上面的都是绝对导入
2.相对导入
    相对导入只能在非主程序的模块中使用
        form .包名/模块名 import 模块/内容
        form ..包名/模块名 import 模块/内容
    因为文件的层级关系，在非主程序的模块去导入其它模块时需要用到 . 或 ..
导入时，解释器会去搜索路径去查找待导入的包/模块
"""
import sys

# 打印搜索路径
print(sys.path)

# 可以自己定义一个路径，加入到搜索路径中
# sys.path.append('e:\\123)
```
