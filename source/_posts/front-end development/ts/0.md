#### 使用工具编写 TS 代码

下载安装 node.js

```bash
npm install -g typescript
```

##### 使用 vscode

创建hello.ts文件，内容如下：

```tsx
console.log("你好");
```

由于`.ts`文件需要编译成`.js`文件才能运行，需要在当前目录执行`tsc 文件名.ts`

##### TS 配置文件

```bash
tsc --init
```

在出现的`tsconfig.json`文件中，可以取消严格模式、编译js的目标版本等

#### TS报错时设置成无法编译

```bash
tsc --noEmitOnError --watch
```
